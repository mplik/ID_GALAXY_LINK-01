ID GALAXY

Interaktywny identyfikator personalny oparty na Arduino Nano — futurystyczna wizytówka dla wydarzeń i targów technologicznych.

Cechy Status

O projekcie

ID_GALAXY_LINK to personalizowany identyfikator zbudowany wokół mikrokontrolera Arduino Nano (ATmega328P). Zamiast tradycyjnej wizytówki urządzenie wykorzystuje wyświetlacz OLED z animacją gwiazd, przycisk funkcyjny oraz kompaktowy, industrialny design.

Główne cechy

  • Sercem projektu: Arduino Nano (ATmega328P)
  • Wyświetlacz: OLED 0.96" I2C z customową animacją gwiazd
  • Zasilanie: Moduł ładowania TP4056 (USB-C) — mobilność i szybkie ładowanie
  • Interakcja: Przycisk funkcyjny uruchamia powitanie i tryby wyświetlania
  • Design: Kompaktowa płytka perforowana z industrialnym sznytem
